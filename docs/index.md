# Wild Jam 26-03 — Art Pipeline

This site documents how to replace the procedural placeholder geometry with
final GLB assets. All 10 models ship as placeholder GLBs generated by
`tools/generate_placeholders.py`; an artist can drop in a replacement GLB at
any time and the game picks it up automatically after a Godot reimport.

shapes exactly.  Keeping your replacement model close to these bounds avoids
needing to adjust CollisionShape sizes in the scene files.

## Quick-start checklist

1. **Generate placeholders** (first-time setup only)

   ```bash
   python tools/generate_placeholders.py
   ```

2. **Open the project in Godot 4.6** — GLBs are auto-imported; `.import` files
   appear in `assets/models/`.

3. **Wire scenes** (first-time only, one-off per scene) — see
   [Wiring Scenes](wiring-scenes.md).

4. **Replace a model** — drop your `.glb` over the existing file, click
   **Reimport** in Godot's FileSystem dock, then run the game.
